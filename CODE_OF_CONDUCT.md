# Contributor Covenant Code of Conduct

All contributors are expected to follow this code of conduct.

## Our Pledge

We pledge to make participation in this project a harassment-free experience for everyone, regardless of background, identity, or beliefs.

## Expected Behavior

- Be respectful and considerate.
- Use welcoming language.
- Show patience with newcomers.
- Focus feedback on code and behavior, not people.

## Unacceptable Behavior

- Harassment, insults, or discriminatory language.
- Threats or abusive behavior.
- Posting private information without permission.
- Other conduct that discourages participation.

## Reporting

If you experience or witness unacceptable behavior, please report it by opening a private security issue or contacting the repository maintainers through GitHub.

## Enforcement

Project maintainers may remove content, block contributors, or take other measures to keep the community safe.
